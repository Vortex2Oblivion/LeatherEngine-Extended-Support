interface IFlxVideoSprite

package hxvlc.flixel

extended by FlxVideoSprite

Interface representing a video sprite object in Flixel.

Variables

autoPause:Bool

Whether the video should automatically pause when focus is lost.

WARNING: Must be set before loading a video.

autoVolumeHandle:Bool

Whether Flixel should automatically adjust the volume according to the Flixel sound system's current volume.

read onlybitmap:Video

The video bitmap.

Methods

load(location:Location, ?options:Array<String>):Bool

Loads a video.

Parameters:

location

The local filesystem path, the media location URL, the ID of an open file descriptor, or the bitstream input.

options

Additional options to add to the LibVLC Media.

Returns:

true if the video loaded successfully, false otherwise.

nextChapter():Void

Moves to the next logical chapter or position in the video, if supported.

pause():Void

Pauses the video.

play():Bool

Plays the video.

Returns:

true if the video started playing, false otherwise.

previousChapter():Void

Moves to the previous logical chapter or position in the video, if supported.

resume():Void

Resumes the video.

stop():Void

Stops the video.

togglePaused():Void

Toggles the pause state of the video.
